Chapter Header

The rubber stamp tool takes a sample of the image, which you can then apply over another image or part of the same image. Each stroke of the tool paints on more of the sampled image. Cross hairs mark the original sampling point.
The pattern stamp tool lets you select part of an image and paint with the selection as a pat- tern. The preview box in the lower right corner of the New Brush dialog box shows the current brush tip.

The preview box in the lower right corner of the New Brush dialog box shows the current brush tip. The box in the lower left corner shows the current brush angle and roundness. As you enter new options, the brushes in these boxes update.
